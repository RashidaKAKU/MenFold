# MenuFold

MenuFold 是一个原生 macOS 菜单栏整理工具，面向刘海屏设计。

## 功能

- 对检测到的菜单栏项目分别设置“始终显示”“折叠”“彻底隐藏”。
- 使用辅助功能扫描，并显示进度、结果与逐项诊断原因。
- 折叠项目显示在原菜单栏下方的独立面板，不进入刘海区域。
- 可设置面板失焦后 1–30 秒自动关闭，或选择永不自动关闭。
- 展开面板可显示项目实时外观，并将点击转发给原项目。
- 可选日期、电池、内存概览，以及 MenuFold 自身的时间显示。
- 支持多显示器、登录时启动和浅色/深色外观。
- 后台扫描不会触发菜单栏拖拽；只在用户修改显示方式时整理一次。

## 构建

```bash
cd MenuFold
./scripts/build-app.sh
open ../outputs/MenuFold.app
```

需要 Swift 6 / macOS 15 SDK 或更高版本。当前工程已在 macOS 27 SDK 下构建。
运行 `./scripts/test.sh` 可执行不依赖 Swift Package Manager 的核心自检。

## 首次使用

1. 在设置页允许“辅助功能”，用于识别、移动和点击菜单栏项目。
2. 给项目选择显示方式；默认全部“始终显示”，不会在首次启动时误隐藏图标。

## 系统边界

macOS 没有公开 API 允许一个应用关闭另一个应用的 `NSStatusItem`。MenuFold 通过系统支持的 Command 拖拽把项目分到显示区和隐藏区，再用真实菜单栏布局空间将隐藏区推入溢出区域。原应用仍在运行，但可见项目会连续重排，不使用图像遮罩或屏幕录制。

少数由系统锁定、禁止移动的项目（例如时钟和控制中心）会保持显示。折叠面板点击和菜单栏整理依赖辅助功能权限。
